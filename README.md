
# AI Resume Screening System 🚀

An AI-powered Resume Screening System built using FastAPI, React, and Machine Learning concepts.

🔗 Live Demo: https://resume-screening-ai-gamma.vercel.app

---

## Features ✨

- Upload Resume PDF
- Paste Job Description
- Resume vs JD Match Score
- NLP Similarity Score
- Extracted Skills Detection
- Matched & Missing Skills
- Smart Recommendations
- Circular Progress Match Indicator
- Live Deployed Frontend & Backend

---

## How It Works ⚡

1. User uploads resume PDF
2. User pastes Job Description
3. Backend extracts resume text
4. NLP compares resume with JD
5. Match score is generated
6. Missing skills & recommendations are shown

## Installation 📦

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
